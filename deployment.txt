--- Deployment 2026-06-26 ---
Initial production deployment of GrooveCart (vinyl record store).
Changes:
- Storefront catalog page (/) with search, genre filter, and sort, backed by GET /api/records.
- Record detail backend GET /api/records/:id.
- Checkout flow: POST /api/checkout (Stripe session + pending order) and OrderConfirmation page polling GET /api/orders/by-session/:sessionId (finalize-on-read, stock decrement).
- Neon Postgres schema (records, orders, order_line_items) synced; 12-record seed present.
- Netlify Recorder integration (audit triggers + backend-request recording endpoint).
- Prices stored as integer cents server-side, mapped to dollars in the API layer.
